**Project Description:**

This project delivers a caregiver-focused mobile assistant for people living with Alzheimer’s or more generally, dementia. It shows the person’s live location on a simple map and sends alerts if they leave a pre-set safe area, helping caregivers act quickly without constant checking. The app also sends medication and routine reminders to both the caregiver and the person, and offers quick mood check-ins—by answering a few simple questions or choosing a picture—so wellbeing can be tracked with minimal effort. A connected smartwatch monitors basic signals like heart rate and movement to highlight possible confusion or distress, and the app includes easy, calming memory games to support daily cognition. A built-in module called MoodAI observes simple behavioral patterns—like daily movement, image choices, and routine adherence—to recognize changes in engagement or mood and surface gentle, timely nudges. Built privacy-first, it keeps data safe with on-device processing where possible, encrypted storage and connections, minimal sharing by default, and clear controls to view, limit, or delete what’s stored.
