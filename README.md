**Project Description:**

This project delivers a caregiver-focused mobile assistant for individuals with Alzheimer’s or dementia. Built in a privacy-preserving way, it provides live location tracking with geofencing alerts, medication and routine reminders for both caregiver and patient, and quick mood check-ins. A connected smartwatch monitors vital signs to detect confusion or distress, and the app includes calming memory games. A built-in MoodAI module observes behavioral patterns to recognize changes in engagement or mood and offers timely nudges. The app prioritizes privacy with on-device processing, encrypted data storage, minimal sharing, and clear user controls.
